Using Interfaces


 An interface is an abstract type that defines members but does not provide an implementation. Interfaces are used to define a contract; a class that implements an interface is expected to adhere to the contract provided by the interface and provide implementations for all the members defined in the interface. Delphi support interfaces. Unfortunately, Pas2JS does not supports interface yet.

type
  TMyObject = class(IMyInterface)
  private
    procedure sayHello;
    function getGreeting: string;
  end;

implementation

procedure TMyObject.sayHello;
begin
  WriteLn('Hello');
end;

function TMyObject.getGreeting: string;
begin

end;
